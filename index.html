Raspberry Pi NTP PPS Server LCD display

This little program uses the Adafruit LCD CharPlate to display the status of a Raspberry Pi NTP server connected to a GPSDO and a 1 PPS source.

Views

There are different views, like clock, number of clients connected, jitter, performance etc. More will be added See video here
